# 3D-Bedroom-Graphics-project-using-OpenGL-3.3

Functions: \
i) Move: Forward, Backward, Left, Right --> W, S, A, D  \
ii) Move: Up, Down --> E, R \
iii) Pitch: Positive, Negative --> 1, 2 \
iv) Yaw: Positive, Negative --> 3, 4 \
v) Roll: Positive, Negative --> 5, 6 \
vi) Fan On/Off --> G\
